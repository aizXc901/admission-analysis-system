# Admission Analysis System

Система анализа поступления для приемной комиссии вуза

## Описание проекта
Веб-приложение для автоматического расчета проходных баллов на образовательные программы с учетом приоритетов абитуриентов.

## Основной функционал
1. Загрузка и обновление конкурсных списков из CSV
2. Визуализация данных с фильтрацией и сортировкой
3. Расчет проходных баллов с учетом приоритетов
4. Формирование PDF+отчетов
5. Отслеживание динамики проходных баллов

## Технологический стек
- **Бэкенд**: Django + Django REST Framework
- **Фронтенд**: React
- **База данных**: PostgreSQL
- **Генерация данных**: Python (Pandas, NumPy)
- **Генерация PDF**: ReportLab/WeasyPrint
