# Система анализа поступления

## Описание проекта

Система анализа поступления - это веб-приложение, разработанное для автоматического анализа конкурсных ситуаций при поступлении абитуриентов в вуз. Приложение позволяет:

- Хранить информацию об абитуриентах, их баллах и приоритетах образовательных программ
- Загружать и обновлять конкурсные списки из внешних источников
- Рассчитывать проходные баллы для различных образовательных программ
- Визуализировать данные в виде таблиц и графиков
- Формировать отчеты в формате PDF

## Технологии

- Python 3.12
- Flask - веб-фреймворк
- Flask-SQLAlchemy - ORM для работы с базой данных
- SQLite - база данных
- Pandas - обработка табличных данных
- ReportLab - генерация PDF-отчетов
- HTML/CSS/JavaScript - клиентская часть
- Chart.js - визуализация данных

## Установка и запуск

### Требования

- Python 3.8+
- pip

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Инициализация базы данных

```bash
python init_db.py
```

### Запуск приложения

```bash
python -m app.main
```

Приложение будет доступно по адресу: http://localhost:5000

## Возможности приложения

1. **Загрузка данных**: Загрузка конкурсных списков из Excel и CSV файлов
2. **Обновление базы данных**: Добавление, удаление и обновление информации об абитуриентах
3. **Расчет проходных баллов**: Автоматический расчет проходных баллов с учетом приоритетов
4. **Визуализация**: Таблицы и графики для анализа данных
5. **Фильтрация**: Фильтрация списков по различным критериям
6. **Генерация отчетов**: Создание PDF-отчетов с полной статистикой

## Образовательные программы

- Прикладная математика (ПМ) - 40 мест
- Информатика и вычислительная техника (ИВТ) - 50 мест
- Инфокоммуникационные технологии и системы связи (ИТСС) - 30 мест
- Информационная безопасность (ИБ) - 20 мест

## Требования к данным

Конкурсные списки содержат следующую информацию:
- Уникальный идентификатор абитуриента
- Наличие согласия о зачислении
- Приоритет образовательной программы (1-4)
- Баллы по различным предметам
- Сумма баллов
